В каких случаях необходима медицинская справка по форме 003 - В/у

Данная справка является обязательным документом необходимым для предъявления в отделение ГИБДД. По сути, это врачебное заключение, свидетельствующее об отсутствии у водителя противопоказании к управлению транспортным средством.
Справка необходима в следующих случаях:

  • Для предъявления в автошколу для получения допуска к экзаменам
  • Для замены прав, срок которых истек
  • Восстановлении водительского удостоверения
  • При открытии новой категории
  • Для профессиональных водителей (при прохождении ежегодного профосмотра)

Порядок прохождения медицинского осмотра для оформления справки 003 в/у в ГИБДД

Обязательным условием для получения справки для ГИБДД является прохождение медицинской комиссии. На начальном этапе водителю необходимо пройти нарколога и психиатра. Затем других специалистов: терапевта, лора, офтальмолога. Офтальмолог в справке может внести дополнительные отметки. По требованию специалистов могут быть назначены дополнительные исследования:

  • ЭКГ
  • Электроэнцефалография
  • Анализ на психотропные и наркотические вещества
